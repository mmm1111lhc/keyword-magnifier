# 🔍 关键词放大镜 — Chrome 扩展

选中网页文字自动放大 + 演示模式 + 中文备注 + 马赛克

## ✨ 功能

| 功能 | 操作 | 效果 |
|---|---|---|
| 🎯 **选中放大** | 鼠标拖选文字 | 文字自动放大 2 倍，保留原色 |
| 🎯 **定点放大** | `Ctrl + 点击` (Mac: `Cmd + 点击`) | 精准放大光标下的单词，专治代码区 |
| 🖥️ **演示模式** | 弹窗开启 | 大鼠标圆圈跟随 + 备注带删除按钮 |
| 📝 **中文备注** | `Alt + 点击` 代码/文字 | 在旁边添加中文注释 |
| 🫥 **马赛克** | 弹窗开启马赛克模式 → 点击文字 | 文字模糊打码，悬停可预览 |
| 🧹 **擦除备注** | 鼠标悬停备注 → 点 ✕ | 逐个删除 |
| 🧹 **全部擦除** | 弹窗点「擦除全部备注」 | 一键清除 |

## 📦 安装

### 方法一：从源码加载（开发者模式）

1. 下载本项目（[Download ZIP](https://github.com/mmm1111lhc/keyword-magnifier/archive/refs/heads/main.zip)）
2. 解压到本地文件夹
3. 打开 Chrome → 地址栏输入 `chrome://extensions`
4. 打开右上角「开发者模式」
5. 点击「加载已解压的扩展程序」
6. 选择解压后的文件夹

## 🎬 使用教程

### 场景 1：演示代码
1. 打开 GitHub 上任意代码文件
2. `Ctrl + 点击` 函数名 → 放大展示
3. 开启「注释模式」→ `Alt + 点击` 代码 → 输入中文解释

### 场景 2：屏幕分享 / 教学
1. 开启「演示模式」→ 鼠标变橙色大光圈
2. 拖选文字 → 自动放大，台下看得清
3. 遇到敏感信息 → 开启「马赛克模式」→ 点击打码

### 场景 3：做笔记
1. `Alt + 点击` 任何文字 → 添加中文备注
2. 不开「演示模式」也能加备注
3. 关闭「显示注释」→ 一键隐藏所有中文
