# 🔍 关键词放大镜 — Chrome 扩展

选中网页文字自动放大 + 演示大鼠标 + 透明画板 + 中文备注 + 马赛克

## ✨ 功能

| 功能 | 操作 | 效果 |
|---|---|---|
| 🎯 **选中放大** | 鼠标拖选 / `Ctrl+点击` | 文字自动放大 2 倍，保留原色 |
| 🖥️ **演示模式** | 弹窗开启 | 大鼠标橙色光圈，演示更清晰 |
| ✏️ **画笔勾画** | 弹窗开启画板 | 透明画板，电容笔/手指在网页上勾画 |
| 📝 **中文备注** | `Alt+点击` 文字/代码 | 在旁边添加中文注释（可删） |
| 🫥 **马赛克** | 弹窗开启 → 点击文字 | 文字模糊打码，悬停预览 |
| 🧹 **擦除** | 悬停备注点 ✕ / 弹窗点全部擦除 | 逐个或一键清除 |

## 🎬 使用场景

### 屏幕分享 / 教学演示
1. 开启**演示模式** → 鼠标变橙色大光圈
2. 开启**画板** → 用鼠标/电容笔在网页上勾画圈点
3. 选中文字 → 自动放大，学生看得清
4. `Alt+点击` → 添加中文备注解释

### iPad / 平板使用
- 安装后开启**画笔模式**
- 用电容笔直接在网页上勾画、圈重点
- 画板是透明的，完全不影响网页操作
- 关掉画板模式，网页恢复正常交互

### 代码演示（GitHub）
1. `Ctrl+点击` 函数名 → 放大展示
2. `Alt+点击` 代码 → 输入中文注释
3. 关闭「显示备注」→ 一键隐藏中文

## 📦 安装

1. 下载本项目 [Download ZIP](https://github.com/mmm1111lhc/keyword-magnifier/archive/refs/heads/main.zip)
2. 解压到本地文件夹
3. Chrome → `chrome://extensions` → 打开右上角**开发者模式**
4. 点击**加载已解压的扩展程序** → 选择解压后的文件夹
